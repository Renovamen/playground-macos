# About This Site

This site is inspired by macOS [Big Sur](https://www.apple.com/in/macos/big-sur/) and [Catalina](https://www.apple.com/bw/macos/catalina/), developed using [React](https://reactjs.org/), [React Redux](https://react-redux.js.org/) and [tailwindcss](https://tailwindcss.com/), and hosted on [Vercel](https://vercel.com/). Some of the icons are generated using [sindresorhus/file-icon-cli](https://github.com/sindresorhus/file-icon-cli).

The source code is hosted [here](https://github.com/Renovamen/playground-macos).
